# OrgMapGenerator
組織図を直感的かつ簡単に作成可能なwebアプリです。
完全ローカル動作です。一切の情報は作成者に収集されません。

# 使用上の注意
自動保存機能はありません。
保存する際は必ず、「ブラウザ保存」か「ファイル書き出し」を行なってください。
「ブラウザ保存」は使用上、ブラウザの更新やキャッシュ削除で意図せず保存したデータが消える可能性があるため、「ファイル書き出し」を強くお勧めします。
